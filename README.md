# toast

一个 toast 弹框插件，遵守 umd 规范

## 直接用\<script>引入

```javascript
<script src="./toast/dist/toast.min.js" />
```

## 模块化引入

```javascript
// es6
import toast from './toast';
// commonjs
const toast = require('./toast');
```

## 使用方式

```js
toast('hi~', 3000); // default 3000
```

## 打包构建

```bash
yarn && yarn build
```
